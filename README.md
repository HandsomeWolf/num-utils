# pixiu-number-toolkit 文档

pixiu-number-toolkit 是一个用 TypeScript 编写的数字工具集。它提供了一组用于数字转换、财务计算、格式化和数学运算的函数。


## 安装

要安装 `pixiu-number-toolkit`，使用以下命令：

```command
npm install pixiu-number-toolkit
```
